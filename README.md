## About The Project

Generic Javascript/NodeJS and HTML/React template to start as a foundation for projects.


database

user (generic user)
repo (name/url and last time accesed and whether currently loaded)
project (associated repo and users)
user-project connect (connect users to project)

use:

get current branch
make modifications and commit/push changes
switch branches
add notes/review comments (with some AI options)
live changes (like google doc)
